# Table Arrangement

| Category | Requires | Version |
|:--------:|:-------:|:--------:|
|Layout > General|API 19, Android 4.4 - 4.4.4 KitKat|1|

## Overview

A formatting element in which to place components that should be displayed in tabular form.
