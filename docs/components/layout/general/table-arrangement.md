# Table Arrangement

| Category | Version | Requires |
|:--------:|:-------:|:--------:|
|Layout > General|1|API 19 | Android 4.4 - 4.4.4 KitKat|

## Overview

A formatting element in which to place components that should be displayed in tabular form.
