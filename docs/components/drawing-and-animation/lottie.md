# Lottie

| Category | Version | Requires |
|:--------:|:-------:|:--------:|
|Drawing and Animation|2|API 19 - Android 4.4 - 4.4.4 KitKat|

## Overview

_A visible component that displays animations from a json or zip file, or from a URL._
