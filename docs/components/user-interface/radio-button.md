# Radio Button

| Category | Requires | Version |
|:--------:|:-------:|:--------:|
|User Interface|API 19, Android 4.4 - 4.4.4 KitKat|2|

## Overview

A visible component that lets the user toggle between two states â€” checked and unchecked. Only one Radio Button in a group of Radio Buttons can be checked at any given time.

## Events

### Changed

Event invoked when the radio button state has been changed.

[[Event('Radio Button', 'Changed', 'checked')]]

| Params | []() |
|--------|------|
|checked|<span class="chip chip-boolean">Boolean</span>|


## Methods

### Toggle

Change the checked state of the view to the inverse of its current state

[[Method('Radio Button', 'Toggle', False)]]
