# Circular Progress

| Category | Version | Requires |
|:--------:|:-------:|:--------:|
|User Interface|1|API 19 - Android 4.4 - 4.4.4 KitKat|

## Overview

_A visible component that indicates the progress of an operation using an animated loop._
