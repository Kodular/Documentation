# Custom Progress

| Category | Requires | Version |
|:--------:|:-------:|:--------:|
|User Interface|API 19, Android 4.4 - 4.4.4 KitKat|4|

## Overview

A visible component that indicates the progress of an operation using a range of indeterminate animations.

## Methods

### Get Styles Names

[[Method('Custom Progress', 'Get Styles Names', True)]]

{>>Returns `list`<<}

Gets style names of all possible animations.
